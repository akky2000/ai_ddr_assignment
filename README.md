# AI-Powered DDR Report Generator

## Overview

This project builds an **AI-powered system that automatically converts inspection and thermal reports into a structured Diagnostic Report (DDR)**.

The system reads raw inspection documents, extracts observations and images, analyzes the data using an AI model, and generates a **client-ready diagnostic report in PDF format**.

This solution demonstrates how AI can automate technical report generation and improve efficiency in inspection workflows.

---

# Problem Statement

Inspection reports and thermal analysis reports often contain large amounts of raw data.

Manually converting these documents into structured diagnostic reports requires significant time and effort.

The objective of this project is to design an **AI workflow that automatically analyzes inspection data and generates a professional DDR report**.

---

# Features

• Upload **Inspection Report (PDF)**  
• Upload **Thermal Report (PDF)**  
• Extract text from PDF documents  
• Extract images from inspection reports  
• AI-based analysis of observations  
• Generate structured diagnostic report  
• Severity assessment and root cause analysis  
• Recommended actions for each issue  
• Final **DDR Report generated as PDF**
